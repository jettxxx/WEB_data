# Upload
    Hướng dẫn upload file có dung lượng lớn lên github
  • Tải về Git LFS và cài đặt: Windowns Linux
  • Mở repo lên:
  • Tạo git: git init
  • Add LFS vào repo: git lfs install
  • Add đuôi: git lfs track "*.zip" "*.rar" "*.exe" "*.gz" "*.msi"
  • Add file config LFS: git add .gitattributes
    • Sau đó có thể add, commit, push các file nặng bình thường :3
